| 카드 클릭 | Detail Modal 표시 |
| "리포트 열람하기" | 전체 페이지 → Report Viewer 전환 |
| "← 목록으로" | Report Viewer → 카드 리스트로 복귀 |
| 카드 호버 | 보더 오렌지 + 그림자 + 미세 리프트 |
| 정렬 변경 | 최신순/오래된순 재정렬 |

                                                              (← 목록으로) → [카드 리스트]
