## 5. Center — 종합 대시보드 탭

### 5-1. 상단 탭 바
- 배경: White, 하단 보더 Gray 200
- 패딩: 0 28px
- 3개 탭: "종합 대시보드" / "경쟁사 비교" / "AI 페르소나"
- 활성 탭: 오렌지 텍스트 + 하단 2px 오렌지 보더
- 비활성: Gray 500, 호버 시 Gray 700
- 우측: 날짜 라벨 "2026.03" 12px Gray 400

### 5-2. KPI 카드 Row (4개 가로 배치)
- 카드: White 배경, 라운드 14px, 보더 Gray 200, 패딩 18px 20px
- 각 카드 구조:
  - 상단: 라벨 (11px Gray 500) + 변화율 뱃지 (10px, 초록/빨강 배경)
  - 수치: 28px ExtraBold (각 카드별 색상)
- 4개 KPI:
  1. 브랜드 인지도 — 78.4% (Navy) ▲ 2.3%
  2. 브랜드 선호도 — 34.2% (Orange) ▲ 1.8%
  3. NPS — +42 (Teal) ▲ 7
  4. 광고 회상률 — 61.3% (#7C3AED 보라) ▲ 4.2%

### 5-3. 차트 영역 (2-Column Grid)
- 좌측: **인지도 추이** — Line Chart (6분기 데이터)
  - X: 24Q1~25Q2
  - 3개 라인: 하나(오렌지), 경쟁A(Navy), 경쟁B(Teal)
  - 카드: White, 라운드 14px, 패딩 20px 24px
- 우측: **NPS 비교** — Bar Chart (수평)
  - 5개 브랜드 비교
  - 오렌지 바

### 5-4. 하단 차트 (2-Column)
- 좌측: **세그먼트 분석** — Radar Chart
  - 5축: 인지도, 선호도, 충성도, 추천의향, 만족도
- 우측: **브랜드 이미지 연상** — Pie Chart

## 6. Center — AI 페르소나 탭

### 6-1. 페르소나 그리드 (3×2)
- 6개 카드, gap 12px
- 카드: White, 라운드 12px, 패딩 16px, 보더 Gray 200
- 호버: 오렌지 보더 + 그림자
- 구조:
  - 아바타 이모지 (32px) + 이름 (13px Bold) + 나이·성별 (11px Gray)
  - 태그 뱃지: 오렌지 배경 #FFF5F3, 오렌지 텍스트
  - 설명: 12px Gray 500

### 6-2. 하나금융 페르소나 6인
1. 👩‍💼 김서연 (28세 여, MZ 직장인)
2. 👨‍👧 이준혁 (45세 남, 가장)
3. 👵 박영순 (62세 여, 시니어)
4. 👨‍💻 최동현 (33세 남, 프리랜서)
5. 👩‍🎓 한소희 (22세 여, 대학생)
6. 🧑‍💼 정민수 (51세 남, 자영업자)

### 6-3. 코웨이 페르소나 6인
1. 👩‍🍳 이하은 (31세 여, 신혼)
2. 👨‍👧‍👦 김태우 (38세 남, 맞벌이 아빠)
3. 👩 박지현 (27세 여, 1인가구)
4. 👵 최순자 (65세 여, 시니어)
5. 🧑‍💼 송민호 (42세 남, 사무실 관리)
6. 👩‍💻 정예린 (29세 여, 건강관리)

### 6-4. 페르소나 클릭 → 인터뷰 패널 (우측 채팅 교체)
- 상단: 페르소나 프로필 카드 (아바타 + 이름 + 태그 + 상세 프로필)
- "인터뷰 시작" 버튼 → 3~4 라운드 Q&A 자동 진행
- Q 버블 (오렌지, 우측) → A 버블 (White, 좌측)
- 타이핑 인디케이터 "응답 중..." 표시
- 완료 후 "인터뷰 완료" 배너
